# 📱 MentorSarthi - AI Mentorship Booking App (React Native)

---

## 🎯 Problem Statement

Finding the right mentor and scheduling sessions can be difficult for users in fast-paced learning environments.  
**MentorSarthi** simplifies this by providing a clean, structured, and intuitive mobile interface to discover mentors and book mentorship sessions seamlessly.

---

## 📱 Overview

MentorSarthi is a React Native (Expo) mobile application built as part of an internship assignment for **Enorvia Global Pvt. Ltd.**

It simulates a real-world mentorship platform where users can browse mentors, view detailed profiles, and book sessions through an interactive flow.

---

## 🚀 Features

### 🔐 Authentication
- Login UI with basic validation  
- Clean and simple user entry flow  

### 🏠 Home Screen
- Personalized greeting section  
- Search functionality UI  
- Horizontal category scroll  
- Mentor listing with modern card design  

### 👨‍🏫 Mentor Details
- Detailed mentor profile view  
- Expertise and rating display  
- Clean information layout  

### 📅 Booking Flow (Core Feature)
- Select available time slots  
- Button enabled only after selection  
- Loading state simulation for booking  
- Confirmation message with selected slot

## 📦 Installation & Setup

Clone the repository:

```bash
git clone https://github.com/sapthapreethi-2006/mentorsarthi-react-native-app.git
```

Install dependencies:

```bash
npm install
```

Start the app:

```bash
npx expo start
```
